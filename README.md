# 🎬 Decision Fatigue - Movie Recommendation App

A Spring Boot application that helps you beat decision fatigue by recommending random movies based on your favorite genres. Inspired by Netflix's "Surprise Me" feature, this project was built to strengthen backend development skills after working primarily in frontend and quality assurance.

## 🎯 Project Origin

As someone who's spent most of my career in **frontend development** and **quality assurance**, I wanted to challenge myself by diving deep into backend development. This project represents my journey into:
- Building RESTful APIs with Spring Boot
- Understanding server-side architecture
- Deploying containerized applications
- Managing API integrations

The Netflix-inspired interface provides a familiar, comfortable experience while the backend handles all the heavy lifting - a perfect blend of my frontend design sensibilities with newly acquired backend skills.

## ✨ Features

- 🎭 **Genre-based Recommendations** - Get random movie suggestions from Action, Comedy, Drama, Horror, Romance, and more
- 🖼️ **Movie Posters** - Automatically displays movie posters from TMDB
- 📱 **Mobile Responsive** - Works seamlessly on both desktop and mobile devices
- 🚀 **Fast & Lightweight** - Built with Spring Boot, optimized for performance
- 🌐 **CORS Enabled** - Ready for cross-origin requests
- 🎨 **Netflix-Inspired UI** - Familiar interface that puts content first

## 🛠️ Tech Stack
