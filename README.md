Decision Fatigue API 🎬📚

Decision Fatigue API is a Spring Boot backend application that helps users make quick content decisions based on mood and time availability.

Instead of endlessly scrolling through movies or books, this API returns a single confident recommendation tailored to the user’s input.

Tech Stack :

- Java 
- Spring Boot
- PostgreSQL (Docker) - might not need database for now
- JPA / Hibernate
- WebClient (for external API integrations – coming soon)
- Swagger (API documentation)
